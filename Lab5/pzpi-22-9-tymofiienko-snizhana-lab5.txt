Короткий опис проекту:
PetHealth - це комплексна програмна система, призначена для
автоматичного моніторингу стану здоров'я тварин, які перебувають у
спеціалізованих готелях для тварин. Система використовує IoT-датчики (смарт-
комірці), що безперервно збирають і передають дані про фізичний стан та
поведінку тварин у реальному часі. Це дозволяє:
- Власникам тварин відстежувати стан своїх улюбленців під час їх
відсутності
- Адміністраторам готелів забезпечувати кращий рівень догляду
- Ветеринарам швидко реагувати на потенційні проблеми зі здоров'ям
Реалізовані функції:
1) Серверна частина (Backend)
- RESTful API для обробки запитів клієнтських додатків
- Авторизація та автентифікація користувачів (JWT)
- CRUD операції для користувачів, тварин та пристроїв
- Зберігання та аналіз даних про здоров'я тварин
- Генерація повідомлень при виявленні відхилень від норми
- Генерація PDF звітів про стан здоров'я тварин
- Інтеграція з MongoDB для зберігання даних
2) Клієнтські додатки
- Веб-застосунок для адміністраторів готелів та ветеринарів
- Мобільний додаток для власників тварин
3) IoT-компонент
Смарт-пристрій (комірець) для збору даних про стан тварин:
- Температура тіла
- Тривалість сну
- Рівень активності3
